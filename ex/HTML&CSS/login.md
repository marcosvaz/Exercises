### A seguinte tela de login deve ser feita com HTML e CSS puros.
![Layout - Dribbble]({{https://cdn.dribbble.com}}/users/905066/screenshots/2648468/dsd.png)
> Você terá uma estrutura da seguinte forma:
```text
container
├── box
│   ├── cabeçalho
│   │   ├── imagem
│   │   └── texto
│   ├── corpo
│   │   ├── texto
│   │   └── formulário
│   │       ├── campo
│   │       └── campo
│   └── rodapé
│       └── botão
└── box:
    ├── título
    └── subtítulo
```
